# Installing and Configuring PostgreSQL on Debian

## add user *postgres*
```
useradd -m -d /home/postgres -s /bin/bash postgres`
```

## initialize db data
```
su -c "su - postgres"  
mkdir /luvade/services/postgresql/data  
initdb -D /luvade/services/postgresql/data
```
